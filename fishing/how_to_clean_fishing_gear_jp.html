釣り道具の手入れ・洗い方

小笠原の釣り道具一式の洗浄・手入れ・整備。

東京都・小笠原諸島2026年6月

01基本の心得

海水は釣り道具にとても厳しいものです。塩分をそのまま乾かすと金属がさび、可動部が固くなり、道具全体の寿命を縮めます。いちばん大切な習慣は、いちばん簡単なことでもあります。毎回の釣行のあと、塩が乾く前に、なるべく早く道具を洗うことです。次に挙げる心得は4種類すべての道具に共通し、その後の各章で道具ごとの詳しい手順を説明します。

  • 真水でやさしく、弱い水流ですすぐ。強い水流は塩を内部に押し込んでしまいます。
  • すすぐ間はドラグを締めたままにし、ドラグワッシャーに水が入らないようにする。
  • 柔らかい布で全体を拭き、そのあと完全に自然乾燥させる。
  • ラインは、しまう前に完全に乾かす（カビ防止のため）。
  • すべて乾いてしまう準備が整ったら、リールのドラグをゆるめる。
  • 直射日光を避け、涼しく乾いた場所で保管する。

02ロッドの手入れ

ガイドとリールシートは、塩やゴミが最もたまりやすい部分です。ここを重点的に手入れします。

  1. リールを外す。 ロッドからリールを外し、ラインを拭き取ります。
  2. ロッドをすすぐ。 竿全体を、弱い水流の真水でやさしくすすぎます。
  3. 洗う。 薄めた中性洗剤で洗い、ガイドとリールシートを特に丁寧にします。
  4. ガイドを磨く。 柔らかい歯ブラシで各ガイドの周りをこすり、たまった塩やゴミをかき出します。あわせてガイドリングのひび割れも確認します。
  5. グリップを洗う。 薄めた洗剤でグリップを洗います。コルクは歯ブラシで、EVAフォームは拭くだけで十分です。
  6. 乾かして保護する。 竿を完全に乾かし、特にリールシートとガイドを念入りに。さらに保護したい場合は、シリコンを軽く吹いた布で金属ガイドを拭きます。
  7. 保管する。 直射日光を避け、涼しく乾いた場所に、できればロッドホルダーやケースに入れて保管します。

03スピニングリールの手入れ

リールはやさしくすすぎ、決して強い水で洗わないこと。高い水圧は塩を内部に押し込んでしまいます。

  1. すすぐ。 釣行直後に、ドラグを締めたまま、弱い水流でやさしくすすぎます。
  2. 洗う。 ぬるめの薄い洗剤水で、塩、皮脂、血、汚れを洗い落とします。
  3. スプール下を清掃。 スプールを外し、芯（スピンドル）とその下を拭きます。
  4. 乾かす。 きれいなタオルで拭き、そのあと自然乾燥させます。
  5. 注油する。 可動部とハンドルに少量、ベアリングには一滴だけ注油します。ごく少量で十分で、つけすぎはゴミを呼び込みます。
  6. 本体を保護。 リール用の防錆剤や軽いつや出し剤を含ませた布で本体を拭きます。
  7. ゆるめて保管。 ドラグをゆるめてしまいます。海水使用なら、年に一度を目安に分解整備に出します。

04ベイトリールの手入れ

ベイトリールはいくつかの部品に分かれるので、塩や砂を洗い流しやすいのが利点です。覚えておくべき唯一のルールは、注油してはいけない場所です。

  1. すすいで拭く。 ロッドから外し、ぬるま湯でやさしくすすぎ、すぐに柔らかい布で拭きます。
  2. 分解する。 サイドプレートを開けてスプールを外します。多くのベイトリールは、本体・ハンドル側のサイドプレート・スプールの3つに分かれます。
  3. 各部品を洗う。 各部品から塩と砂を洗い落とし、各ベアリングを綿棒で清掃します。
  4. 完全に乾かす。 組み立て直す前に、すべての部品を完全に乾かします。
  5. 適切な箇所に注油。 各スプールベアリング（キャストコントロールキャップの下、パームサイドカバーの内側、スプールシャフト）、ハンドルベアリング、レベルワインド、ウォームギアに一滴ずつ注油します。スタードラグにも注油し、開け閉めしてなじませます。
  6. 避ける箇所を知る。 ブレーキ機構とワンウェイクラッチには注油しないこと。ここに油がつくと不具合の原因になります。
  7. 組み立てて保管。 組み直し、ドラグをゆるめて保管します。海水使用なら、半年に一度を目安にしっかり整備します。

05伸縮式ランディングネットの手入れ

網と柄では手入れが異なります。網は腐らないようにすすいで乾かし、柄はスムーズに伸縮するようロック部に塩を残さないことが大切です。

  1. よくすすぐ。 たっぷりの真水で網と柄をすすぎ、塩、ぬめり、うろこ、砂を洗い流します。網を広げる、または吊るすと、水が網全体に行き渡ります。
  2. 必要なら洗う。 汚れがひどいときは、薄めた中性洗剤で洗います。
  3. 柄を洗う。 柄の各段を、薄めた洗剤水とスポンジで洗います。
  4. 網を完全に乾かす。 たたんで保管する前に、網を完全に乾かします。濡れたままだとカビや臭い、コードの腐食につながります。
  5. ロック部の手入れ。 ロック部を真水ですすぎ、塩で固着しないようにします。各段をシリコン布で拭くと、滑りが良くなり水もはじきます。
  6. 点検して保管。 網の破れ、枠やロック部のさびを確認します。乾いた状態で保管し、長時間の直射日光を避けます。

06ライン（釣り糸）

ラインは見落とされがちですが、ロッドやリールよりも切れることが多い部分です。塩、紫外線、擦れで少しずつ弱るため、毎回の釣行後に手早く点検します。

  • PE（編み糸）：GT、カンパチ、マグロ、ハタ類を狙う道糸。
  • フロロカーボン：道糸に結ぶリーダー。
  • ナイロン：エサ釣りで使うことがある。

毎回の釣行後

  1. 巻いたまますすぐ。 ラインをリールに巻いたまま、スプールを真水でやさしくすすぎます。
  2. 乾かす。 しまう前にラインを完全に乾かします。
  3. 先端を点検。 先端から5〜10mを、ほつれ、つぶれ、擦れ、変色がないか確認します。
  4. 切って結び直す。 傷んだ部分を切り、結び直します。
  5. 結びを確認。 すべての結び目がしっかり締まっているか確かめます。
ライン交換の目安
PE（編み糸）6〜12か月
フロロのリーダー傷んだら交換
ナイロン約6か月

07フック（針）

針は重要な消耗品です。海水ですぐにさび、先が鈍ったりさびたりすると魚を逃すため、毎回乾かして点検します。

  • サークルフック：生きエサ、リリース向き。
  • Jフック：一般的なエサ釣り。
  • アシストフック：ジギング。
  • トレブルフック：GTルアー、ポッパー、スティックベイト。
  1. すすぐ。 毎回、真水でしっかりすすぎます。
  2. 乾かす。 箱に戻す前に完全に乾かします。
  3. 先を確認。 爪に引っかからない先は鈍っています。
  4. 研ぐか交換。 鈍った先は軽く研ぎ、さびた針はすぐ捨てます。
  5. 乾燥保管。 乾いたタックルボックスに、乾燥剤（シリカゲル）と一緒に保管します。

08ルアー

小笠原の釣りはジギングとキャスティングが中心なので、ルアーにも専用の手入れが必要です。塩は本体よりも先に、針・スプリットリング・金具をいためます。

  • メタルジグ：カンパチ、マグロ、イソマグロ。
  • スロー系ジグ：オナガなど深場の魚。
  • ポッパー／スティックベイト：GT、マグロ。
  • インチクジグ：アカハタや根魚。
  1. 針を外す。 ルアーから針を外します。
  2. すすぐ。 真水ですすぎます。
  3. 乾かす。 完全に乾かします。
  4. リングを確認。 曲がったりさびたスプリットリングは交換します。
  5. 本体を確認。 塗装と目の傷みを確認します。
  6. スイベルに注油。 内蔵スイベルに一滴だけ注油し、回転を保ちます。

09仕掛け（ターミナルタックル）

仕掛け類は小さく見落としがちですが、さびたスイベルやひ弱なスナップ一つで良い魚を逃します。ほかと同じようにすすいで点検します。

  • スイベル（サルカン）：糸ヨレを防ぐ。
  • スナップ：ルアー交換を素早く。
  • スプリットリング：針とルアーをつなぐ。
  • オモリ：エサを狙いの深さに保つ。
  • ビーズ：結び目を保護する。
  • サビキ仕掛け：ウメイロ釣りに重要。
  1. すすぐ。 すべての仕掛けを真水ですすぎます。
  2. 乾かす。 完全に乾かします。
  3. 点検。 さびを確認し、スナップやスイベルが滑らかに動くか確かめます。
  4. 交換。 さびた部品は、本番で壊れる前に交換します。

10魚を扱う道具

GT、カンパチ、イソマグロなど小笠原の大物は、フィッシュグリップ、プライヤー、ナイフで扱います。海水とぬめりにさらされるので、洗浄と軽い注油が欠かせません。

  • フィッシュグリップ：大物を安全に保持（GT、カンパチ、イソマグロ）。
  • プライヤー：針外し、スプリットリング交換、リーダー切り。
  • ディフッカー（針外し器）：深く飲まれた針を外す。
  • ナイフ：魚をさばく。
  1. すぐ洗う。 使用後すぐにグリップ、プライヤー、ナイフを洗います。
  2. しっかり乾かす。 すべての道具を完全に乾かします。
  3. 関節に注油。 プライヤーやグリップの可動部に一滴注油します。
  4. ナイフの手入れ。 刃に薄く油をぬり、鞘に収めて保管します。
  5. 動作を確認。 バネ式のアゴやカッターが効くか確認し、さびを落とします。

11エサの保存用具

小笠原の釣りはイカ、ムロアジ、貝類、ウニに頼るため、エサを新鮮に保つ用具は清潔で臭わない状態に保ちます。

  • バッカン（エサバケツ）：生きエサや切りエサを入れる。
  • エアポンプ：生きエサを生かす。
  • ライブウェル：大きめの活かし水槽。
  • 保冷クーラー／氷箱：エサと釣果を冷やす。
  1. 空にする。 残ったエサやぬめりを出します。
  2. 洗う。 洗剤と水で洗います。
  3. 除菌する。 臭いと雑菌を抑えるため除菌します。
  4. 乾燥。 すすいで完全に自然乾燥させてから保管します。

12安全装備

小笠原の磯は外洋に面した高い火山岩で、安全装備は必須です。塩と紫外線で劣化するので、ほかの道具と同じようにすすぎ、乾かし、点検します。

  • ライフジャケット：必須。
  • 磯靴（スパイク／フェルト）：滑りやすい火山岩で必須。
  • 偏光サングラス：魚や地形を見やすく、目を守る。
  • グローブ：糸・リーダー・魚から手を守る。
  • 救急セット：必須。
  1. すすいで乾かす。 ライフジャケットとグローブを真水ですすぎ、直射日光を避けて完全に乾かします。
  2. ライフジャケットを点検。 バックルとベルト、膨張式ならガスボンベを確認します。
  3. 磯靴の手入れ。 すすいで乾かし、スパイクやフェルト底の摩耗を確認します。
  4. サングラスを拭く。 きれいな布で拭き、ケースに入れて保管します。
  5. 救急セットを補充。 中身を補充し、使用期限を確認します。

14タックルの保管と持ち運び

小笠原へ道具を無事に運ぶことは、洗うのと同じくらい大切です。清潔で乾いた、保護された状態で運びます。

  • ロッドケース（チューブ）：移動中のロッドを守る。
  • リールカバー：傷やぶつけを防ぐ。
  • 防水タックルボックス：仕掛けを乾いた状態で守る。
  • ドライバッグ：スマホ、書類、電子機器を守る。
  1. 清潔・乾燥で詰める。 すべて清潔で完全に乾いた状態にしてから収納します。
  2. ドラグをゆるめる。 保管・移動の前にすべてのドラグをゆるめます。
  3. 涼しく保管。 直射日光を避け、涼しく乾いた場所で保管します。
  4. 長期保管。 3か月以上保管するなら、リールから古いラインを外すことを検討します。

分解整備について

リールの完全な分解整備は細かく、小さな部品をなくしやすい作業です。内部がジャリジャリしたり動きが渋いと感じたら、自分で開けるよりも釣具店に整備を任せるのが安心です。
