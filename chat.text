Tokiq — Social Precision Timing Game
Product Vision

Design a modern mobile-first gaming platform called Tokiq.

Tokiq is a real-time multiplayer social game where players compete by stopping a timer as close as possible to a visible target time.

The game combines:

Human timing skill
Anticipation
Competition
Social interaction
Jackpot rewards

The experience should feel like a blend of:

A live game show
A multiplayer competition
A social casino game
An esports-style event

without looking like a traditional gambling application.

The design should be elegant, premium, modern, and highly engaging.

Core Game Mechanic

At the start of each round:

A target time is announced.

Examples:

7.25 seconds
9.80 seconds
11.45 seconds

The target is visible to everyone.

Players join the round by contributing a small entry fee.

A countdown begins.

When the round starts:

A timer begins counting upward.

Players must press a large STOP button when they believe the timer has reached the target time.

The player closest to the target wins.

Example:

Target = 10.00s

Player A = 9.98s
Player B = 10.03s
Player C = 10.11s

Player A wins.
